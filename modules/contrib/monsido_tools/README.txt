Introduction
------------

All-in-one platform that unifies web governance, quality assurance, and
accessibility compliance. Monsido help you optimize your website, quickly
and easily.

Accessibility laws vary by country and sector, so Monsido ensure that your
site is validated against the international standard, the WCAG 2.1. Make
your website accessible to everyone, including people with disabilities
, and achieve global accessibility compliance. Monsido scans your site to
 find any possible issues that may hinder accessibility, giving you an overview
of how your website is doing, as well as recommendations on how to address them.

Installation
------------

Install the module as you would normally install a
contributed Drupal module. Visit https://www.drupal.org/node/1897420 for
further information.
-------------

This module requires no modules outside of Drupal core.


Configuration
--------------
   1. Navigate to Administration > Extend and enable the monsido Tools module.
   2. Configure the module at admin/config/monsido/settings- add your token
      and clicking "Save" button.

Support
-------

If you experience a problem with Monsido Tools or have a problem, file a request
or issue on the flag queue at http://drupal.org/project/issues/monsido_tools.
